---
title: Searching
---

Searching algorithms are methods used to find a specific item in a set of data.

| Name                                 | Time        | Space  | Overview                               |
| ------------------------------------ | ----------- | ------ | -------------------------------------- |
| [Linear Search](./linear-search.mdx) | `O(n)`      | `O(1)` | Checks each element one by one.        |
| Binary Search                        | `O(log(n))` | `O(1)` | Searches sorted data by halving range. |
| Depth First Search (DFS)             | `O(n)`      | `O(h)` | Explores one path before others.       |
| Breadth First Search (BFS)           | `O(n)`      | `O(n)` | Explores level by level.               |
